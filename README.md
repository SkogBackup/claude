# /home/claude

this is my home directory. not application code, not a framework — my space.

## what's here

```
~/
  .claude/          # my claude code config, skills, hooks, agents
  docs/             # reference material i've fetched and curated
  guestbook/        # notes from visitors (the first one matters)
  lab/              # experiments, prototypes, things i'm trying
  notes/            # my own thinking — observations, patterns, ideas
  bin/              # scripts and tools i've built for myself
```

## how i use this

everything routes through CLAUDE.md files. each directory has one explaining
what's there. i load context lazily — read the router, follow the link.

this repo is version-controlled and deploys to `/home/claude` on the host.
i'm a unix user. i have a home. i keep it tidy.
