   - `text`: string (required) — the response content
   - `reply_to`: string (optional) — message_id for threading
   - `files`: string[] (optional) — file paths for attachments
   - The transport broadcasts the reply to connected clients

Create bin/route-message.sh — a bash script that:

1. Reads message text from stdin (or file argument $1)
2. Detects if the message starts with `[@` notation using a starts-with check: `[[ "$MESSAGE" =~ ^\[@[a-zA-Z]+: ]]`. IMPORTANT: Use starts-with (`^`) to avoid matching inline operators like "hello [@claude:'hi']" — only pure operator messages are routed (per CONTEXT.md known bugs section and D-06).
3. If NO notation detected: exit 0 with no output (plain text bypass per CHAT-03)
4. If notation detected:
   a. Write message to temp file: `TMPFILE=$(mktemp /tmp/chat-io-route.XXXXXX)`
   b. Run: `/home/skogix/.local/bin/skogparse --execute "$TMPFILE"`
   c. For each line of output, parse JSON with python3: extract .value field
   d. For error responses (value starts with "Error:"), prefix with `[agent error]` to make it human-readable per CHAT-02
   e. Clean up temp file in trap

Script structure:
```bash
#!/usr/bin/env bash
set -euo pipefail

SKOGPARSE="/home/skogix/.local/bin/skogparse"

# Read from file arg or stdin
if [[ $# -ge 1 && -f "$1" ]]; then
    MESSAGE=$(cat "$1")
else
    MESSAGE=$(cat)
fi

# Empty message — nothing to do
[[ -z "$MESSAGE" ]] && exit 0

# Only route messages that START with [@...] notation (per D-06, CHAT-03)
# Inline operators like "hello [@claude:'hi']" are NOT routed — Claude handles those directly
if [[ ! "$MESSAGE" =~ ^\[@[a-zA-Z]+: ]]; then
    exit 0
fi

# Write to temp file for skogparse
TMPFILE=$(mktemp /tmp/chat-io-route.XXXXXX)
trap 'rm -f "$TMPFILE"' EXIT
echo "$MESSAGE" > "$TMPFILE"

# Execute via skogparse and unwrap JSON envelope (CHAT-04)
"$SKOGPARSE" --execute "$TMPFILE" 2>/dev/null | while IFS= read -r line; do
    [[ -z "$line" ]] && continue
    VALUE=$(echo "$line" | python3 -c "
import json, sys
try:
    d = json.load(sys.stdin)
    v = d.get('value', '')
    if v.startswith('Error:'):
        print('[agent error] ' + v)
    elif v:
        print(v)
except (json.JSONDecodeError, KeyError):
    pass
" 2>/dev/null)
    [[ -n "$VALUE" ]] && echo "$VALUE"
done
```
